# کارگاه سیم و چوب — نمونه‌کار طراحی وب

صفحه‌ی معرفی یک کارگاه فرضی سازسازی سنتی؛ HTML/CSS/JS خالص، بدون فریم‌ورک.

## ساختار پروژه
```
index.html
style.css
script.js
assets/images/   ← عکس‌های واقعی را اینجا بگذار (اسم فایل‌ها پایین‌تر)
```

## عکس‌ها
سایت طوری ساخته شده که تا وقتی عکس واقعی نگذاشتی، به‌جای خالی یا خراب‌شدن، یک رنگ/طرح جایگزین نشان می‌دهد. کافیست فایل‌هایی با همین نام‌ها را داخل `assets/images/` بگذاری:
- `workshop.jpg` — عکس کارگاه یا میز کار
- `tar-1.jpg`, `setar-1.jpg`, `tar-2.jpg` — عکس هر ساز در بخش نمونه‌کارها

## انتشار روی گیت‌هاب (GitHub Pages)
1. یک ریپازیتوری جدید بساز (مثلاً `siym-o-choob`)
2. همین فایل‌ها را داخلش push کن
3. برو به Settings → Pages → زیر Branch گزینه‌ی `main` را انتخاب کن و Save بزن
4. بعد از حدود یک دقیقه، سایت روی آدرسی مثل زیر بالا می‌آید:
   `https://hasanam16.github.io/siym-o-choob/`
